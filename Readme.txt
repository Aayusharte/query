# Company Database SQL

This repository contains an SQL script to create a simple company database with tables for `Departments` and `Employees`. The database includes various constraints like `PRIMARY KEY`, `NOT NULL`, `DEFAULT`, `AUTO INCREMENT`, `UNIQUE`, and `FOREIGN KEY`.

### Tasks:
- Create a `CompanyDB` database.
- Create `Departments` and `Employees` tables with relevant constraints.
- Insert sample data into the tables.

### How to Use:
2. Open MySQL Workbench (or any MySQL client).
3. Run the SQL script (`company_database.sql`) to create the database and tables.
4. Use SQL queries to interact with the database.

### Example Queries:
- Select all employees and their department names.
- Update an employee's salary.
- Delete an employee record.
